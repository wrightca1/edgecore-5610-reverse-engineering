## OpenNSL Mapping Notes

OpenNSL 3.5.0.1 uses `opennsl_*` naming. Key mappings:
- bcm_l3_route_add -> opennsl_l3_route_add
- bcm_l2_addr_add -> opennsl_l2_addr_add
- bcm_vlan_create -> opennsl_vlan_create
- bcm_port_enable_set -> opennsl_port_enable_set
- bcm_init / bcm_attach -> opennsl_init / opennsl_driver_attach

Cumulus uses Broadcom SDK 6.3.8 (proprietary), not OpenNSL. API surface is similar but parameter structures may differ.
